# 产品裁剪与页面映射

## 一期建议保留的一级导航

- 首页
- 知识库
- FAQ
- 智能问答
- 共享空间
- 系统设置

其中“系统设置”仅平台管理员可见。

## 一期建议隐藏或弱化的能力

- Agent 独立入口
- MCP 配置入口
- Web Search 默认开关
- GraphRAG 相关入口
- 面向普通用户的高级模型与推理参数

## 后续阶段保留能力清单

这些能力一期不作为前台主路径开放，但应保留在架构和代码层，作为二期或三期演进储备：

- Agent 能力与自定义智能体
- MCP 外部工具接入
- Web Search
- GraphRAG 与知识图谱相关能力
- 高级模型配置与多模型切换
- 多数据源同步与更丰富的 connector
- 多渠道 IM 接入
- 更复杂的智能体编排与工具链调用

## 二期/三期保留能力清单

这些能力当前以“隐藏前台入口或不纳入一期主路径”的方式保留，不做删除：

- Agent 能力与自定义智能体编辑
- MCP 服务配置与外部工具调用
- Web Search 外部检索能力
- GraphRAG 与知识图谱增强能力
- 高级模型切换与调试参数
- 多数据源自动同步能力
- 多渠道 IM 接入能力

## 现有页面映射

- `knowledge-bases`：保留，作为知识库主入口
- `knowledge-search`：保留，作为搜索入口
- `creatChat` 与 `chat/:chatid`：保留，产品表达改为“智能问答”
- `organizations`：保留，产品表达改为“共享空间”
- `settings`：保留，但要做管理员化和分组收敛
- `agents`：一期从前台主路径移除

## 首批前端改造重点

- 从侧边栏移除 Agent 主入口
- 从用户菜单移除 Ollama、Web Search、MCP 快捷入口
- 从设置页移除一期不开放的配置项
- 把页面表达从“AI 平台”收敛为“知识库系统”
