# 角色权限矩阵

## 角色定义

- 普通用户：消费知识
- 编辑者：维护知识内容
- 空间管理员：管理共享空间与成员
- 平台管理员：维护平台配置与运行

## 权限矩阵

| 功能 | 普通用户 | 编辑者 | 空间管理员 | 平台管理员 |
|---|---|---|---|---|
| 搜索知识 | 是 | 是 | 是 | 是 |
| 发起问答 | 是 | 是 | 是 | 是 |
| 查看引用来源 | 是 | 是 | 是 | 是 |
| 查看知识库 / FAQ | 是 | 是 | 是 | 是 |
| 创建知识库 | 否 | 是 | 是 | 是 |
| 上传文档 / 在线录入 | 否 | 是 | 是 | 是 |
| 编辑 FAQ | 否 | 是 | 是 | 是 |
| 重解析文档 | 否 | 是 | 是 | 是 |
| 共享知识库 | 否 | 受限 | 是 | 是 |
| 邀请成员 / 审批加入 | 否 | 否 | 是 | 是 |
| 管理空间设置 | 否 | 否 | 是 | 是 |
| 模型 / 存储 / 检索配置 | 否 | 否 | 否 | 是 |
| 系统状态查看 | 否 | 否 | 否 | 是 |
| MCP / GraphRAG 配置 | 否 | 否 | 否 | 谨慎 |

## 一期权限原则

- 普通用户只面向“搜”和“问”
- 编辑者只面向“传”和“改”
- 空间管理员只面向“共享”和“成员管理”
- 平台管理员只面向“平台配置”和“运行维护”
