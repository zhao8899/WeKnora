# 知识图谱专家测试流程评审（已归档）

**原始日期：** 2026-04-22  
**当前状态：** 已归档  
**归档原因：** 原文是知识图谱专家链路的专项技术评审；GraphRAG/知识图谱生产级能力当前不作为一期上线阻塞项。

## 保留结论

原评审中仍需保留的判断：

- 当前知识图谱专家链路更接近“图谱感知的混合搜索”，不应在一期产品中被包装为完整 GraphRAG 闭环。
- 如果只是导入文档、选择知识图谱专家、提问并得到看似合理答案，不能证明图谱关系查询已经完整生效。
- 知识图谱能力应作为二期或专项增强方向，避免干扰一期知识库主链路。

## 当前产品口径

一期不把 GraphRAG 生产级能力作为普通租户主路径。普通租户应优先完成：

- 知识库建设。
- FAQ 维护。
- 搜索和问答。
- 来源查看和答案可信度判断。

GraphRAG、知识图谱专家、复杂 Agent 工具链进入二期候选能力池。
