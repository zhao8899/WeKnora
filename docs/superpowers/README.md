# Superpowers 资料说明

本目录保留历史专项方案、技术评审和研发计划的归档摘要。

## 当前定位

`docs/superpowers` 不是当前项目排期来源，也不是一期上线检查清单。这里的文档主要用于追溯历史技术判断和能力演进背景。

当前有效资料以以下文档为准：

- [WeKnora Roadmap](../ROADMAP.md)
- [企业内部知识库底座一期](../企业内部知识库底座一期/README.md)
- [当前待办与上线检查](../企业内部知识库底座一期/05-任务分解与上线检查.md)
- [角色权限矩阵](../企业内部知识库底座一期/06-角色权限矩阵.md)

## 维护规则

- 新的当前任务不要写入本目录。
- 已归档文档只保留摘要、当前结论和承接位置。
- 如需查看原始任务步骤、技术细节或历史讨论，通过 Git 历史追溯。
- 涉及一期范围、角色、优先级、上线检查的内容，应更新到 `docs/企业内部知识库底座一期/` 或 `docs/ROADMAP.md`。
